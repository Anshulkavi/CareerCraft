# CareerCraft 🎯

**CareerCraft** is an all-in-one career development platform designed to empower job seekers by offering tools like a Resume Builder, Job Matcher, AI Assistant (coming soon), and more. It simplifies the job-hunting journey by providing smart, personalized, and professional career services in one place.

---

## 🚀 Features

### 📝 Resume Builder
- Create professional resumes with dynamic sections
- Multiple templates including Modern and Simple layouts
- Download as PDF using `@react-pdf/renderer`
- Section customization (Add/Delete Skills, Experience, etc.)

### 💼 Job Matcher
- Personalized job suggestions based on user profile
- Smart filters: Skills, location, experience
- Bookmark & track applications (coming soon)

### 🤖 AI Career Assistant *(Coming Soon)*
- Resume improvement suggestions
- Career-related Q&A
- Mock interview generator *(Planned)*

### 📊 Dashboard *(Planned)*
- Visualize job application stats
- Track resume downloads and updates

---

## 🧱 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Lucide Icons
- @react-pdf/renderer
- React Router DOM

### Backend
- Django + Django REST Framework
- MongoDB (via Djongo)

### Other Tools
- JWT Authentication
- Vite (for dev speed)
- React Context API
